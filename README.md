## ゲームの現状

- 現在、ゲームの最初の部分（NPCとの会話部分）のみが実装されています。
- 「プレイヤー」カードを左クリックし、「ガイド」カードの上にドラッグ＆ドロップすると会話が開始されます。
- その後、「ガイドのスーツ」を「プレイヤー」の上にドラッグ＆ドロップし、所持させた後、再度「ガイド」の上にドラッグ＆ドロップすることで、追加の会話が行われます。
- 現在、会話の後に続く部分は開発中です。今後のアップデートで追加される予定です。
